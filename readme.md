## 关于选课

### 注意事项

> [!CAUTION]
>
> 1. **<u>不允许跨段选课。</u>**<br>每学期会被分为上下两段，请大家按照课表所示时间在对应周数内选课。跨段选课可能会导致分数录入异常从而影响最终成绩。

2. **选满规定数量的实验。**<br>少选实验，对应分数记为0。先保证自己有实验可做，再去考虑实验和老师怎么样。因个人原因错过实验时间导致旷课的（特殊情况请联系老师，老师联系方式在系统课表里有），请在系统中补选对应数量的实验。

> [!TIP]
>
> 3. **注意关注系统。**<br>系统开放选课的时间不定，大家注意经常关注系统。
>    
>   貌似实验开始前任意时刻均可退选与选课，保险起见与避免实验资源浪费，强烈建议实验开始24小时内不要退选。

4. **选课前仔细检查空闲时间。**<br>选课系统不会自动检查课程冲突。注意实验时间不要和课程与考试冲突了。

## 关于正课（实验）

### 前期准备

买好需要的纸。校内各教育超市均有售。<br>需要用到的纸张包括：**物理实验专用报告纸、物理实验专用数据记录纸**、方格纸（示波器实验有些老师会要求用到，作图按需选用），如下图所示。<br>一般来说，一次实验平均会用到2张报告纸、2张记录纸，大家可根据实验数量推算张数。方格纸若无作图需要买一张小的即可，以防万一可以买两张小的或一张大的。

![6c6c54b7a05897eacbc57f590d98ab09](./readme.assets/6c6c54b7a05897eacbc57f590d98ab09.jpeg)

### 课前预习

按照上述5.2中的操作方法（[点击此处跳转](# 系统使用指南)），下载讲义、PPT等资料。系统中的课表包含教师联系方式，大家记住就好。

根据实验资料（讲义/PPT），完成（抄）预习报告。部分实验的报告样例在本仓库/网盘链接中已经给出（[点击此处跳转](# 写在前面)）。小红书和部分学院公众号等平台也有报告样例，大家可做参考。

一份合格的预习报告应该包含以下部分：<br>**实验目的、实验原理、实验仪器、实验内容（包含步骤）**

**差不多两个格子即可（报告纸正面写满）**，如果比较多可以多抄一点，不同老师标准不一样，有些觉得完整一点比较好，有些觉得概括一点比较好。实验编号就是系统上的“实验台”编号，报告柜号由班长/学委提供。

### 去上课

看好课表，务必带好校园卡，提前到达实验室。<br>**不要掐点！不要掐点！不要掐点！**晚1秒刷卡也是迟到！迟到一次总成绩-0.5！

建议再带好一沓空白数据记录纸和草稿纸备用。

到达教室，在**教室前方刷卡机刷卡签到**。桌子上有贴编号，对号入座。桌上有文件夹，里面是讲义和PPT（公共，请勿带走）。<br>手机放入包中，存放至指定位置。拿出预习报告和水笔（示波器实验还要拿格子纸和铅笔橡皮尺子），按老师要求向老师提交预习报告。<br>可以把空白数据记录纸、草稿纸和餐巾纸、水杯、计算器也一并拿出来备用。<br>为了实验安全，水杯请放在地上。

![image-20260313223724607](./readme.assets/image-20260313223724607.png)

上课时间到，开始实验。前面先听老师讲解，然后自己独立动手操作，有问题问老师，视情况也可求助周围同学。

> [!CAUTION]
>
> **注意实验数据必须用水笔记录，不允许使用修正带和铅笔！**（示波器作图除外）

记错了请划去，尽量不要涂改。

做完必做实验，将写好的数据记录纸交给老师检查。老师检查通过后完成选做实验或进行数据处理与分析。若完成选做实验，做完后记得务必找老师再次检查选做实验结果！（选做实验数据旁边标一下“选做”两个字）

### 提交报告

一周内将实验报告和数据记录纸提交至实验室门口的报告柜。数据记录纸夹在实验报告里面以免丢失。交的时候注意看清报告柜格子上对应的**实验**和**时间**以免交错。

实验结束后，老师打完分后会同步在系统中显示。有的老师批的快，有的老师批的慢，不建议主动去催老师。<br>但结课考试前3天若系统中仍没有成绩，请联系老师。老师联系方式在系统实验资料的小课表中。

## 如何写好一份报告

### 除了努力之外的技巧

除了努力、把字写好写整齐之外，下面的小技巧也能帮助你写出一份好的报告：

1. **报告结构明确清晰**<br>每个部分区分清楚，各个实验的分析不混在一起。
2. **正确绘制图表**<br>一张图表应该包含**图名**、**图例**（单图例可不画）、作图区（**数据点**与**拟合曲线(若有)**）、**坐标轴（刻度间距合理）**及对应**物理量名称**与**单位**，若有拟合直线应给出**直线方程（表达式）**与**相关系数（R值）**。缺一不可。
   参考示例：
   ![image-20260314154159580](./readme.assets/image-20260314154159580.png)
3. **给出完整且正确的计算过程**<br>计算过程应给出公式、带入数值。不确定度计算可参考绪论PPT，下面7.2（[点击此处跳转](# 需要用到的前置知识)）给出了部分内容。  参考示例：
   ![image-20260314155600834](./readme.assets/image-20260314155600834.png)
4. **正确取舍有效数字**<br>理论上，计算时需要严格遵循有效数字取舍法则（下面7.2（[点击此处跳转](# 需要用到的前置知识)）给出了取舍方法）。但由于Excel自身的特性，同时也是为了计算方便，我们可通过尽可能在计算中间过程保留精确性的方法，仅在最后一步进行有效数字的取舍。

## 关于考试

### 考试形式

闭卷考试，时间60分钟。题型为选择题、判断题、填空题。根据实验不同，部分实验只设客观题。考试时会给出全部实验的题目，按照要求选择规定数量的实验作答即可。上学期6个，下学期6个，医科为5个。可以选没做过的实验。

### 一些小技巧

提前看看讲义和PPT。因为可以选没做过的实验，所以可以建议看看自己没做过的实验，特别是比较简单的。
